### Soil Data Access Training Resources
Repository: [Soil-Data-Access-Training-Resources](https://github.com/jneme910/Soil-Data-Access-Training-Resources)

A practical learning resource for using **Soil Data Access with SQL and APIs**, including real-world SSURGO examples, AOI workflows, and interpretation queries for conservation planning and soil analysis.

### NASIS Pedons
Repository: [NASIS-Pedons](https://github.com/jneme910/NASIS-Pedons)

ArcGIS tools that query pedon data from the **National Soils Information System (NASIS)** and compile spatial datasets for analysis and mapping.

### Soil Application Metrics
Repository: [Soil-Application-Metrics](https://github.com/jneme910/Soil-Application-Metrics)

Documentation and logic for **web soil survey metrics**, usage analysis, and reporting workflows.

### SVI Nutrient Runoff
Repository: [SVI-Nutrient-Runoff](https://github.com/jneme910/SVI-Nutrient-Runoff)

A soil sensitivity and nutrient-runoff modeling project combining **SSURGO, elevation derivatives, LiDAR-informed terrain analysis, and SQL-based interpretation logic**.
